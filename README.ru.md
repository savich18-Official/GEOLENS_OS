### <p align="center">Продвинутая Платформа OSINT-Разведки и Форензики</p>

**ГЕОЛИНЗА ОС** — это высокоуровневая платформа для цифровой криминалистики и разведки по открытым источникам (OSINT), созданная для триангуляции точного местоположения по визуальным данным. Используя нейронный движок Gemini 3.1 Flash, система анализирует тончайшие улики окружения для обхода физических границ и идентификации геолокации с хирургической точностью.

## ⚡ Основная Миссия
> "Информация — это ключ, контекст — это оружие."

Обеспечение точных разведданных о местоположении путем перекрестного сопоставления филиалов глобальных брендов, архитектурных маркеров и биологического контекста. Больше никаких «местоположение неизвестно» — только подтвержденные координаты, подкрепленные форензик-уликами.

## 🚀 Ключевые Особенности

*   🔍 **Глубокая Триангуляция**: Анализ видов деревьев, схем покраски бордюров и региональных вариаций столбов электропередач.
*   🏢 **Кросс-референс Филиалов**: Определение того, находится ли бизнес в **Ялте** или **Ставрополе**, путем анализа специфических интерьерных планов и форензик-идентификаторов.
*   🕰️ **Исторический Контекст**: Сопоставление визуальных изменений во времени для обеспечения максимально точного совпадения.
*   📄 **Форензик-Отчеты**: Генерация детальных отчетов в стиле спецагентов сразу в терминале.

## 🛠️ Технологический Стек

| Компонент | Технология | Роль |
| :--- | :--- | :--- |
| **Логика** | `React 18 & Vite` | Высокоскоростной нейро-движок |
| **ИИ Узел** | `Gemini 3.1 Flash` | OSINT-рассуждения и форензика |
| **Стиль** | `Tailwind CSS` | Интерфейс в стиле хакерского терминала |
| **Анимация** | `Framer Motion` | Плавные состояния и частицы |
| **Безопасность** | `Шифрование` | Отсутствие логов, защищенные потоки |

## 💻 Развертывание Системы

```bash
# Клонировать репозиторий
git clone https://github.com/savich18-Official/geolens-os.git

# Войти в узел
cd geolens-os

# Установить зависимости
npm install

# Инициализировать соединение
npm run dev
```
