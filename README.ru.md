# 🌐 ГЕОЛИНЗА ОС V1.0 - Продвинутый OSINT-Исследователь

![Hacker Aesthetic](https://i.redd.it/sgoydbg7yfo11.gif)

**ГЕОЛИНЗА ОС** — это высокоуровневая платформа для цифровой криминалистики и разведки по открытым источникам (OSINT), созданная для триангуляции точного местоположения по визуальным данным. Используя нейронный движок Gemini 3.1 Flash, система анализирует тончайшие улики окружения для обхода физических границ.

---

## ⚡ Основная Миссия
Обеспечение точных разведданных о местоположении путем перекрестного сопоставления филиалов глобальных брендов, архитектурных маркеров и биологического контекста. Больше никаких «местоположение неизвестно» — только подтвержденные координаты.

## 🛠 Технологический Стек
- **Фронтенд Движок**: React 18 & Vite (Высокоскоростная среда выполнения)
- **Нейронный Процессор**: Google Gemini 3.1 Flash-Preview (Специализированные OSINT промпты)
- **Визуальный Интерфейс**: Tailwind CSS & Motion (Кинематографичный хакерский UI)
- **Шифрование**: Встроенные защищенные потоки данных с отсутствием локального хранения.
- **Иконки**: Lucide-React (Интерфейсные глифы)

## 📡 Протокол Локализации
Система поддерживает более 16 языков нейронного интерфейса для глобальной доступности:
- **Основные узлы**: Английский, Русский, Японский, Китайский, Украинский, Белорусский, Казахский, Испанский.
- **Расширенные узлы**: Португальский, Польский, Голландский, Вьетнамский, Финский, Итальянский, Французский, Немецкий.

## 🚀 Ключевые Особенности
- **Глубокая Триангуляция**: Анализ видов деревьев, покраски бордюров и вариаций столбов электропередач.
- **Кросс-референс Филиалов**: Определение того, находится ли бизнес (например, квест-комната) в Ялте или Ставрополе на основе специфических визуальных форензик-маркеров.
- **Исторический Контекст**: Учет изменений окружающей среды на протяжении времени существования локации.
- **Форензик-Отчеты**: Генерация детальных отчетов в стиле спецагентов в формате Markdown.
