# 🌐 GEOLENS OS V1.0 - Investigador OSINT Avançado

![Hacker Aesthetic](https://i.redd.it/sgoydbg7yfo11.gif)

**GEOLENS OS** é uma plataforma de forense digital e inteligência de fontes abertas (OSINT) de alto nível, projetada para triangular localizações exatas a partir de dados visuais. Equipado com o motor neural Gemini 3.1 Flash, o sistema analisa pistas ambientais sutis.

---

## ⚡ Missão Principal
Fornecer inteligência de localização precisa através da referência cruzada de filiais de marcas globais, marcadores arquitetônicos e contexto biográfico. Chega de "localização desconhecida" — apenas coordenadas confirmadas.

## 🛠 Stack Tecnológico
- **Frontend**: React 18 & Vite
- **Processador Neural**: Google Gemini 3.1 Flash-Preview
- **Interface**: Tailwind CSS & Motion (UI Hacker)
