# 🌐 GEOLENS OS V1.0 - Просунутий OSINT-Дослідник

![Hacker Aesthetic](https://i.redd.it/sgoydbg7yfo11.gif)

**GEOLENS OS** — це високорівнева платформа для цифрової криміналістики та розвідки за відкритими джерелами (OSINT), розроблена для тріангуляції точного місцезнаходження за візуальними даними. Завдяки нейронному двигуну Gemini 3.1 Flash, система аналізує найменші підказки оточення для обходу фізичних кордонів.

---

## ⚡ Основна Місія
Забезпечення точних розвідданих про місцезнаходження шляхом перехресного зіставлення філій глобальних брендів, архітектурних маркерів та біологічного контексту. Більше жодних «місцезнаходження невідоме» — тільки підтверджені координати.

## 🛠 Технологічний Стек
- **Фронтенд Двигун**: React 18 & Vite
- **Нейронний Процесор**: Google Gemini 3.1 Flash-Preview (Спеціалізовані OSINT промпти)
- **Візуальний Інтерфейс**: Tailwind CSS & Motion (Кінематографічний хакерський UI)
- **Шифрування**: Вбудовані захищені потоки даних.

## 📡 Протокол Локалізації
Система підтримує понад 16 мов інтерфейсу для глобальної доступності.

## 🚀 Ключові Особливості
- **Глибока Тріангуляція**: Аналіз видів дерев, фарбування бордюрів та варіацій стовпів.
- **Кросс-референс Філій**: Визначення філії бізнесу (наприклад, квест-кімнати) на основі специфічних форензік-маркерів.
- **Форензік-Звіти**: Генерація детальних звітів у стилі спецагентів.
