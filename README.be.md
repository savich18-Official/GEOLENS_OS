# 🌐 ГЕАЛІНЗА ОС V1.0 - Прасунуты OSINT-Даследчык

![Hacker Aesthetic](https://i.redd.it/sgoydbg7yfo11.gif)

**ГЕАЛІНЗА ОС** — гэта высокаўзроўневая платформа для лічбавай крыміналістыкі і разведкі па адкрытых крыніцах (OSINT), створаная для трыянгуляцыі дакладнага месцазнаходжання па візуальных дадзеных. Выкарыстоўваючы нейронны рухавік Gemini 3.1 Flash, сістэма аналізуе найтанчэйшыя ўлікі наваколля.

---

## ⚡ Асноўная Місія
Забеспячэнне дакладных разведдадзеных аб месцазнаходжанні шляхам перакрыжаванага супастаўлення філіялаў глабальных брэндаў, архітэктурных маркераў і біялагічнага кантэксту. Больш ніякіх «месцазнаходжанне невядома» — толькі пацверджаныя каардынаты.

## 🛠 Тэхналагічны Стэк
- **Фронтэнд Рухавік**: React 18 & Vite
- **Нейронны Працэсар**: Google Gemini 3.1 Flash-Preview
- **Візуальны Інтэрфейс**: Tailwind CSS & Motion

## 🚀 Ключавыя Асаблівасці
- **Глыбокая Трыянгуляцыя**: Аналіз відаў дрэў, афарбоўкі бардзюраў і варыяцый слупоў.
- **Крос-рэферэнс Філіялаў**: Вызначэнне філіяла бізнесу па форензік-маркерах.
