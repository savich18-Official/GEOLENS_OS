## ⚡ Core Mission
To provide precise locational intelligence by cross-referencing global brand branches, architectural markers, and biological context. No more "location unknown" — only confirmed coordinates.

## 🛠 Tech Stack
- **Frontend Engine**: React 18 & Vite (High-speed runtime)
- **Neural Processor**: Google Gemini 3.1 Flash-Preview (Custom OSINT Prompting)
- **Visual Interface**: Tailwind CSS & Motion (Cinematic Hacker UI)
- **Encryption**: Built-in secure data streams with zero local persistence.
- **Icons**: Lucide-React (Interface Glyphs)

## 📡 Localization Protocol
The system supports over 16 neural interface languages to ensure global accessibility:
- **Major nodes**: English, Russian, Japanese, Chinese, Ukrainian, Belarusian, Kazakh, Spanish.
- **Extended nodes**: Portuguese, Polish, Dutch, Vietnamese, Finnish, Italian, French, German.

## 🚀 Key Features
- **Deep Triangulation**: Analyzes tree species, curb paint, and utility pole variations.
- **Multi-Branch Cross-Referencing**: Identifies if a business (like a quest room) is in Yalta or Stavropol based on specific interior forensic markers.
- **Historical Context**: Considers environmental changes over the lifespan of a location.
- **Forensic Reporting**: Generates detailed "Special Agent" style reports in Markdown.

## 💻 Development & Deployment
1. **Clone the Repo**: `git clone https://github.com/savich18-Official/geolens-os.git`
2. **Inject Dependencies**: `npm install`
3. **Establish Connection**: `npm run dev`
4. **Access Node**: `http://localhost:3000`
