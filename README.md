## ⚡ Core Mission
> "Information is the key, context is the weapon."

To provide precise locational intelligence by cross-referencing global brand branches, architectural markers, and biological context. No more "location unknown" — only confirmed coordinates validated by forensic evidence.

## 🚀 Key Features

*   🔍 **Deep Triangulation**: Analyzes tree species, curb paint patterns, and regional utility pole variations.
*   🏢 **Multi-Branch Cross-Referencing**: Identifies if a business is in **Yalta** or **Stavropol** by analyzing specific interior layouts and forensic identifiers.
*   🕰️ **Historical Context Engine**: Cross-references visual changes over time to ensure the most accurate match.
*   📄 **Forensic Reporting**: Generates comprehensive "Special Agent" style reports in Markdown directly from the terminal.
