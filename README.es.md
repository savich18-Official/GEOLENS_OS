# 🌐 GEOLENS OS V1.0 - Investigador OSINT Avanzado

![Hacker Aesthetic](https://i.redd.it/sgoydbg7yfo11.gif)

**GEOLENS OS** es una plataforma de análisis forense digital y de inteligencia de fuentes abiertas (OSINT) de alto nivel, diseñada para triangular ubicaciones exactas a partir de datos visuales. Con el motor neuronal Gemini 3.1 Flash, el sistema analiza pistas ambientales sutiles para superar fronteras físicas.

---

## ⚡ Misión Principal
Proporcionar inteligencia de ubicación precisa mediante la comparación cruzada de sucursales de marcas globales, marcadores arquitectónicos y contexto biológico. No más "ubicación desconocida" — solo coordenadas confirmadas.

## 🛠 Stack Tecnológico
- **Motor Frontend**: React 18 & Vite
- **Procesador Neuronal**: Google Gemini 3.1 Flash-Preview (Prompts OSINT personalizados)
- **Interfaz Visual**: Tailwind CSS & Motion (UI de hacker cinemática)

## 🚀 Características Clave
- **Triangulación Profunda**: Analiza especies de árboles, patrones de pintura en bordillos y variaciones en postes de servicios públicos.
- **Referenciación Cruzada de Sucursales**: Identifica si un negocio está en Yalta o Sebastopol basado en marcadores forenses interiores específicos.
